Visual Artist Guide for AI

Welcome to the visual reference tool for the Anima 2B parameter model.

This library features over 20,000+ artist tags based on the Danbooru dataset. Our explorer provides visual benchmarking for Stable Diffusion and ComfyUI users, showcasing how the model interprets different artistic styles, from classic masters to modern illustrators.

Core Features

  • 20,000+ Artist Styles: Comprehensive visual database generated with optimized parameters in the Anima 2B model.
  • High-Resolution Previews: Click on any card to view the generation in full detail. Use left/right keys to navigate.
  • Smart Sorting:
    • Name: Alphabetical navigation.
    • Uniqueness: Artists sorted by how distinct their visual fingerprint is.
    • Works: Sorted by number of works in the original dataset (popularity).
  • Favorites System: Click the heart to curate your personal prompt book. Export and import your curation as JSON across devices.
  • Text Export: Quickly export a raw `.txt` list of your favorite artists to drop straight into ComfyUI wildcards.
  • Search & Jump: Filter by specific artist names or jump to specific popularity ranks instantly.

Technical Specs

All images were generated purely via the prompt: 1girl, artist_name to ensure the model's base interpretation of the style is untainted by complex prompt engineering. CFG and steps were locked to standard benchmarks.
